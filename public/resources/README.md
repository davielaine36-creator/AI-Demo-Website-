# Intake questionnaire PDF

Place the Client AI Systems Intake Questionnaire PDF here and name it:

```
client-ai-systems-intake-questionnaire.pdf
```

The Intake page (`/intake`) links to `/resources/client-ai-systems-intake-questionnaire.pdf`
via a "Download Full Intake PDF" button.

- Anything in the `public/` folder is served as-is at the site root, so the
  final URL will be `https://your-domain/resources/client-ai-systems-intake-questionnaire.pdf`.
- If the file is missing, the site still builds and runs — the download button
  simply 404s. The short web form on the Intake page works either way.

> A starter copy of the questionnaire is already included in this folder. Replace
> it any time you update the document (keep the same filename).
